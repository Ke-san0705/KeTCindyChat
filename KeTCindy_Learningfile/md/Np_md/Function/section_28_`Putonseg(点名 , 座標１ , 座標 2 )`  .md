# 平面の図形とグラフ  
## 描画  
### 点・線分・直線  
#### `Putonseg(点名 , 座標１ , 座標 2 )`  
`Putonseg(点名 , 座標１ , 座標 2 )`  
線分上に点を作る  説明  
座標１，座標 2 を端点とする線分上に点名の点を作る。できた点は線分上しか動かない。指定した点がすでに存在する場合は動かさない。  
【例】  
線分 AB 上に点 C をとる。  
```  
Putonseg("C",A,B);  
```  
点 (-1,0),(2,2) を通る線分上に点 C をとる。  
```  
Putonseg("C",[[-1,0],[2,2]]);  
```
